# Checklist de despliegue

## Antes de publicar

- npm run lint OK.
- npm run build OK.
- Revisión visual en local.
- Revisión responsive.
- Enlaces activos revisados.
- CV Español abre.
- GitHub, LinkedIn, Credly y contacto funcionan.
- Demo RematePOS funciona.
- No hay rutas locales.
- No hay .env.
- No se sube dist ni node_modules.

## GitHub

- Rama feature actualizada.
- Pull Request hacia main.
- CI de GitHub Actions OK.
- Revisión final.
- Merge a main.

## Vercel

- Proyecto conectado a GitHub.
- Framework Vite.
- Build Command npm run build.
- Output Directory dist.
- Preview Deployment revisado.
- Producción desde main.
- URL pública revisada.

## Post-deploy

- Revisar Home.
- Revisar Proyectos.
- Revisar RematePOS.
- Revisar Contacto.
- Revisar móvil.
- Revisar CV.
- Revisar enlaces externos.
